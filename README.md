#SEO Updates
========

Download the files on your local host or web server.

Run index.php

Enter any website URL and you can get the following informatio of the website:

  1. Meta Keywords
  2. Meta Description
  3. Title Tag
  4. IP Address
  5. Load Time //also depens on your internet connection
  6. HTTP Status
  7. Internal Links
  8. External Links
